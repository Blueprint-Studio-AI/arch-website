What is Arch?

Arch is a Bitcoin-native VM, chain, and validator network.

Together they execute on Bitcoin UTXOs, and settle assets back to Bitcoin’s ledger.

Native tech

What’s built on Arch lives on Bitcoin:
same coins, same wallets, same ledger.

Making that true took new technology at almost every layer:

  • Direct Bitcoin settlementassets are never wrapped
  • UTXO-native executionprograms run on real UTXOs, not IOUs
  • Per-account Bitcoin addressesevery balance verifiable on any explorer
  • Per-account UTXOsfunds are never pooled into one wallet
  • Taproot account modelcompatible with existing Bitcoin wallets
  • Threshold Schnorr signing (FROST + ROAST)no single key ever exists, signing never stalls
  • Native Bitcoin syscallsprograms read and write Bitcoin directly, no oracle in between
  • Bitcoin-native VMSolana’s proven eBPF engine, forked and rebuilt around UTXOs
  • Fast dPoS consensus180ms blocks, 1,500 TPS

Financial primitives

Speed, reliability, and general programmability become financial infrastructure on Bitcoin.

This is where the native tech pays off. Pooling on real UTXOs enables AMMs. Oracle price feeds enable collateral enforcement. Enforceable collateral makes lending safe, and safe lending carries credit, perps, and structured products.

  • Pooled liquidity
  • Oracle price feeds
  • Collateral enforcement
  • Liquidation engine
  • Token issuance
  • Risk & margin

Finance unlocks
